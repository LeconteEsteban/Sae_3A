Installation du Projet ESCAPE

Disclamer

Nous considérons que nous nous adressons un informaticien, donc nous ne détaillerons pas l'intégralité de l'installation, par exemple nous ne détaillerons pas l'installation de POSTGRESQL. Nous considérons que seule une personne compétente lira notre guide.

Penser à lire notre README

Pour optimiser les performances, nous mettont en cache beaucoups de data

Pré-requis

- Une connexion à internet

- POSTGRESQL avec l'extension VECTOR.

- Taille du projet. Minimum : 100mo; Maximum : 10go (dépend de l'installation complète des requirements, importation des modèles d'embedding )

- Taille de la bdd: ~250mo

1. Cloner le dépôt

Clonez le projet depuis GitHub :

git clone https://github.com/LeconteEsteban/Sae_3A.git

2. Installer les dépendances

requirements.txt se situe dans python/worker

pip install -r requirements.txt

3. Configurer la connexion à la base de donnée POSTGRESQL

Il faut créer un fichier ".env" à partir du fichier ".env_exemple"

                DB_HOST=localhost
                DB_PORT=port
                DB_NAME=nom_bdd
                DB_USER=mon_utilisateur
                DB_PASSWORD=mon_mot_de_passe

4. Mettre en place le back python

Il faut executer python/db_create_fill.py , ce script permet de créer la base de données, et de la remplir

uvicorn main:app --reload

5. Démarrer le serveur

uvicorn main:app --reload

Accédez à votre API sur http://127.0.0.1:8000/docs
